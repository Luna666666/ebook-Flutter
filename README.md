# ebook-Flutter
## ebook-Flutter是什么?
一个基于Dart语言开发的高内聚低耦合的基于谷歌Flutter移动跨平台开发学习项目
## ebook-Flutter有哪些功能？

* `今日`功能
* `预警`功能
* `数据`功能
* `我的`功能

## 更新记录
2019.4.17 -- v1.0.0:提交1.0.0版本并完成框架搭建

## 使用方式
1.0 下载DiseaseControl直接运行

## 注意事项
1. 熟悉Swift4.0基本语法
